# ESP-Byte
## ESP32-S3-mini + Ebyte nRF24 multi-purpose controller PCB 
### Features/components:
- ESP32-S3-mini-1U connected via SPI to Ebyte E01 2G4M27SX (nRF24)
- flash ESP32 via USB-C; 
- Power from USB-C or lithium battery
- BQ2409 lithium battery charger IC with switch to toggle between 1A and 257mA fast-charge rate
- Resistor divider to monitor battery level
- Buck-boost converter to maintain 3.3V output across battery life
- Joystick, buttons, extra GPIOs for UART, I2C OLED, power, etc.
